## Setup Instructions

### Step 1: Clone the repository

```sh
git clone https://github.com/chanakya-ex3/ChanakyaBhardwajMudunuru_21BCE1010.git
cd ChanakyaBhardwajMudunuru_21BCE1010
```
### Step 2: Setup Backend
```sh
cd backend
npm i
nodemon index.js
```

### Step 3: Setup frontend
```sh
cd frontend
npm i
npm run dev
```
